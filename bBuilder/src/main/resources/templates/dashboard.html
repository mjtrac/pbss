Quick Start

⚡ Quick Election Setup

Create a complete uniform election — one ballot for everyone — in a single form. Sets up the election, ballot combinations, and design template in one step. Best for simple general elections with no party variants.

Manual Election Setup (complete steps 1–5 in order)

1. Elections

Create an election and set its type and date.

2. Regions

Create SinglePrecincts, then PrecinctGroups (Districts, Cities, etc.) and assign precincts to them.

3. Parties

Pre-seeded with "Everyone" for general/nonpartisan elections. Add party-specific entries for primaries.

4. Ballot Types

Precinct, Mail-In, Absentee, Provisional — pre-seeded for you.

5. Contests & Candidates

Add races and measures, assign each to regions, then add candidates.

Ballot Configuration (complete steps 6–7 before printing)

6. Ballot Combinations

Define each unique ballot variant: Election + Precinct + Party + Ballot Type. Shows which contests will appear on each ballot.

7. Design Templates

Set paper size, vote indicator style (oval/checkbox), columns, fonts, and barcode position for each election.

Ballot Printing

Print Ballot

Select a ballot combination and generate a PDF for a voter. Requires steps 1–7 above to be complete.

Sample Data

Add Sample Contests & Print

Adds cheese and federal contests to the initial sample election, assigns them to all precinct groups, and triggers ballot generation. Safe to run more than once.
