What this creates automatically:
  1. A Party named All (if one doesn't exist for this jurisdiction)
  2. A SinglePrecinct region named All (if one doesn't exist)
  3. A BallotType named Precinct (if one doesn't exist)
  4. The Election with the name you entered
  5. A Ballot Design Template with the layout settings above
  6. A Ballot Combination (Election + All region + All party + Precinct type)
  7. A placeholder Contest named "EditMe" with two candidates "EditMe1" and "EditMe2" — you will be taken directly to this contest to rename it

After saving: rename the contest and candidates, set the voting method and max choices, then go to Print Ballot and select the combination to test your ballot.
